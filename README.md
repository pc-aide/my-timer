# 2x Timer Chrome Extension

A simple Chrome extension that provides **two timers side by side** for easy tracking of multiple tasks or time intervals.

## Features

- Display **two countdown timers** in the same popup.
- Set **custom durations** for each timer.
- Timers run **simultaneously** and independently.
- **Persistent settings**: your timer durations are saved locally in the browser.
- Optional **notifications** when a timer ends.

## How to Use

1. Click the 2x Timer icon in the Chrome toolbar.
2. Set the duration for **Timer 1** and **Timer 2**.
3. Click **Start** to begin counting down.
4. Reset or pause timers at any time.
5. The timers will retain their last set durations even after closing the browser.

## Permissions

- **Storage**: Used to save timer settings locally.
- **Notifications** (optional): Used to alert you when a timer finishes.

## Privacy

This extension does **not collect, transmit, or share any personal data**. All timer settings are stored locally in the user's browser and are never sent to any server.

## Installation

1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the folder containing this extension.
